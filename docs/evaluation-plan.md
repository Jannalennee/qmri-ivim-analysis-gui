# Evaluation Plan (DSR Phase 3)

## Setting
Amsterdam UMC case-study context with representative stakeholders.

## Participants
Target: 5-10 users

- Clinician-researchers
- PhD researchers
- IT / technical staff

## Task Scenarios

1. Import scan dataset
2. Configure model for selected workflow
3. Run qMRI estimation
4. Interpret maps and uncertainty
5. Export result snapshot

## Metrics

- Task completion success
- Completion time per task
- Interaction errors / recoveries
- Qualitative interview feedback

## Analysis

- Quantitative: descriptive statistics (mean, standard deviation)
- Qualitative: thematic analysis (usability, trust, workflow fit)

## Deliverables

- Usability summary report
- Prioritized improvement backlog
- Versioned changes linked to findings
